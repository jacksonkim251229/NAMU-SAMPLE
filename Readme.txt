------------------------------------
OpenAPI 사용 설명서 및 예제 파일 안내
------------------------------------

1. 충분한 검증 후 자료(설명서 및 예제)를 제공해 드리고 있습니다만 그럼에도 불구하고
  의도하지 않게 잘못된 내용이 포함될 수 있으니 이 점 감안하시어 충분한 점검 후
  자료를 활용해 주시기 바랍니다.


2. 아래 순서대로 자료를 살펴보시는 것을 권장합니다.

  1) SDK.doc        : WMCA 소개 및 함수 안내
  2) FAQ.doc        : WMCA를 활용시 자주 물어보시는 내용들
  3) 시세_SPEC.doc  : 시세 및 투자정보 TR 및 실시간
  4) 주문_SPEC.doc  : 주문 및 계좌관련 TR 및 실시간
  5) SAMPLES        : 예제


3. 예제의 자료 입출력 구조(I/O)가 맞지 않는 경우 (1) SDK.doc 문서 마지막 페이지에 기술되어있는 
'파일로그' 기능을 이용하여 wmca.log 파일을 생성하여 분석할 수 있습니다.
분석을 통해서도 원인을 찾을 수 없을 경우 wmca.log 파일을 당사로 보내주시면 분석을 지원해드리겠습니다


4. 과거 제공되었던 wmcaTransact() 함수는 추후 서비스 중단될 예정이오니 wmcaQuery() 함수로 교체하여 이용하시기 바랍니다.

  BOOL wmcaQuery(HWND hWnd, int nTRID, constchar* szTRCode, const char* szInput, int nInputLen, int nAccountIndex)


5. 2025년 3월 4일부터 국내 주식 종목에 대해 '대체거래소' 제도가 시작되었습니다.
이로 인해 한국 거래소(KRX)만 존재하던 기존 환경에서 몇 가지 변화가 있습니다.

  ex) 스마트주문(SOR) 및 통합시세

오랜 기간 KRX 시장에 맞도록 프로그램을 개발/이용해오신 분들께서는 
새로운 변화에 맞게 프로그램을 반드시 수정하여 이용하시기 바랍니다.

  1) 기존 서비스(TR 및 실시간 전문)들은 2026년에 단계적으로 서비스가 중단될 예정입니다.
  2) wmca 기존 버전을 사용하셨던 분들은 반드시 변경된 버전으로 모듈을 교체하시기 바랍니다.  
  (신규 모듈은 압축 파일 bin 폴더에서 확인 가능)
